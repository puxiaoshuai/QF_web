em相对于父元素的字体大小来说的

rem相对于html的元素来说的

优点，通过修改html中文字大小来改变页面元素的大小

媒体查询 @media 可针对不同的屏幕尺寸来设置不同的样式

        @media mediatype and|not|only (media feature){
            css 代码
        }

mediatype 分为 all 所有 print 打印 scree 电脑

media feature 分为 width min-width max-width

不同屏幕要进行写不同的样式，需要使用不用的样式表 在link是 使用 media="mediatype and oonxxxx

建议:媒体查询最后从小到大进行判断
