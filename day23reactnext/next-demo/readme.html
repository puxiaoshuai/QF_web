钩子事件

routeChangeStart->路由开始变化,

routeChangeComplete->路由结束变化

routeChangeError->跳转发生错误,

beforeHistoryChange->在改变浏览器 history之前触发

hashChangeStart->hash跳转开始时执行

hashChangeComplete->hash跳转完成时
