根据jk ts学习

day1

  1. 初始化项目 npm init -y 一路默认
  2. npm i typescript -g 全局安装ts，
  3. tsc --init 进行 文件配置tsconfig.json配置
  4. tsc xx.ts能把ts文件编译成js

工程话，npm i webpack webpack-cli webpack-dev-server -D

复制配置文件后，安装相关的包, npm i ts-loader typescript html-webpack-plugin clean-webpack-plugin webpack-merge -D

其中插件的含义:html-webpack-plugin :该插件将为你生成一个 HTML5 文件， 在 body 中使用 script 标签引入你所有 webpack 生成的 bundle

在package.json中进行命令配置，就能运行项目和 打包项目了。
